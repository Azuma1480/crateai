# CrateAI

CrateAI is a mobile-first record library and DJ preparation app for analog DJs.

## Project layout

- `crate-ai/` - Active React, Vite, and PWA application source.
- `crateai.html` - Original standalone prototype retained for project history.

## Development

```powershell
cd crate-ai
npm install
npm run dev
```

Run `npm run build` before publishing changes.
